# Customer_Behaviour_Analysis_Project
Data analytics project showcasing customer behavior analysis using python, sql, and power Bi. 




## 📊 Data Analytics Project

## 📌 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to delivering actionable insights through dashboards and reports. The goal is to extract meaningful patterns, improve data quality, and present findings in a clear, business-friendly format.

---

## 📂 Dataset

* Source:  Kaggle
* Format:  CSV 
* Description: The dataset contains information related to customer shopping behavior.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas) – Data loading, cleaning, and EDA
* **SQL** (PostgreSQL) – Data querying and transformation
* **Power BI** – Interactive dashboard creation
* **Gamma** – Presentation (PPT) creation
* **Jupyter Notebook** – Development environment

---

## 🔄 Project Workflow

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Verified structure, data types, and initial records

### 2. Exploratory Data Analysis (EDA)

* Checked distributions, trends, and relationships
* Identified missing values and outliers
* Visualized key metrics using charts and graphs

### 3. Data Cleaning

* Handled missing values
* Removed duplicates and inconsistencies
* Standardized formats and data types

### 4. SQL Analysis

* Loaded cleaned data into relational databases:

  * PostgreSQL
  
* Wrote queries for:

  * Aggregations (SUM, AVG, COUNT)
  * Joins and filtering
  * Business KPI extraction

### 5. Dashboard Development

* Built an interactive **Power BI dashboard**
* Included:

  * Key performance indicators (KPIs)
  * Trend analysis
  * Filters and slicers for user interaction

### 6. Reporting & Presentation

* Created a structured analytical report summarizing insights
* Developed a presentation using Gamma to communicate findings effectively

---

## 📊 Dashboard Highlights

* Overview of key metrics and KPIs
* Trend analysis over time
* Category-wise or segment-wise performance
* Interactive filtering for deeper insights

---

## 📈 Results & Insights

* Identified key trends and patterns in the dataset
* Highlighted performance drivers and potential issues
* Provided actionable recommendations based on analysis

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/mpsuryamp/Customer_Behaviour_Analysis_Project.git
cd your-repo-name
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run Python Analysis

* Open Jupyter Notebook:

```bash
jupyter notebook
```

* Execute the notebook step-by-step

### 4. Set Up Database

* Import cleaned dataset into: PostgreSQL
* Run SQL scripts from `/sql` folder

### 5. View Dashboard

* Open the `.pbix` file in Power BI Desktop

### 6. Presentation

* Open the Gamma presentation link/file included in the project
